# 《小云雀AI探险记》课程总览

> 本系列共32课，面向3-9岁儿童，以团团（7岁男孩）和点点（云朵形AI机器人）为主角，通过趣味故事讲解人工智能基础知识。每课约3分钟。

---

## 角色介绍

| 角色 | 描述 |
|------|------|
| **团团** | 7岁男孩，黑灰短发，圆脸腮红，橙色上衣，蓝色背带裤，好奇活泼 |
| **点点** | 白色云朵形AI机器人，深色屏幕脸，青蓝发光眼睛，胸口爱心屏幕，头顶紫色天线 |

---

## 课程目录与核心知识点

### 第一部分：基础概念（第1-9课）

| 课次 | 标题 | 主题 | 核心知识点 |
|------|------|------|------------|
| 第1课 | 小侦探找线索 | 信息是什么 | 能帮助我们找到答案、做出判断的线索，就是信息；多条信息组合，判断更准确；AI思维：输入信息→分析线索→做出判断 |
| 第2课 | 开关城的两兄弟 | 0和1是什么 | 0代表"关/没有/停下"，1代表"开/有/运行"；多个0和1排队，可以表示、存储和传递更多信息 |
| 第3课 | 小邮差的密码信 | 编码 | 编码 = 给消息"穿外套"变成别人看不懂的样子；解码 = 把外套脱掉恢复原消息；编码和解码必须用同一套规则 |
| 第4课 | 仓库管理员机器人 | 存储 | 存储 = 把信息放进格子里；格子 = 记忆位置；容量 = 格子的数量 |
| 第5课 | 小方块拼出大世界 | 像素 | 图片由很多小方块（像素）组成；像素越多，分辨率越高，画面越清晰 |
| 第6课 | 红绿蓝灯的合唱团 | RGB颜色 | RGB三原色：红(Red)、绿(Green)、蓝(Blue)；红+绿=黄，绿+蓝=青，红+蓝=紫，红+绿+蓝=白 |
| 第7课 | 行李箱装不下啦 | 压缩 | 无损压缩（整理但不丢失）vs 有损压缩（去掉不重要的）；压缩太多会丢失重要细节 |
| 第8课 | 雾气里的照片 | 噪声 | 噪声是干扰信号，让信息变模糊；垃圾进，垃圾出——错误数据导致错误结果 |
| 第9课 | 红灯停绿灯行 | 条件判断 | 条件判断根据"如果...那么..."决定做什么；多个条件叠加让规则更复杂 |

### 第二部分：算法基础（第10-17课）

| 课次 | 标题 | 主题 | 核心知识点 |
|------|------|------|------------|
| 第10课 | 会数数的刷牙机器人 | 循环 | 循环 = 重复执行同一个步骤；两种方式：数着次数 / 做到为止；电脑最擅长做重复的事 |
| 第11课 | 做披萨的流水线 | 任务分解 | 流水线 = 把大工作分成一步一步；串行 = 按顺序执行；并行 = 同时做不同的事 |
| 第12课 | 找bug的小修理工 | 调试 | Bug是隐藏的小错误；调试四步骤：发现问题→找到原因→修改→再试 |
| 第13课 | 袜子大整理 | 分类 | 分类 = 按特征把相似的东西归到一起；分类规则不同，结果就不同；好的特征能区分事物 |
| 第14课 | 特征放大镜 | 特征 | 特征 = 可以用来区分事物的特点；找特征就像玩"找不同"游戏；好的特征 = 有区分度的特征 |
| 第15课 | 猜一猜明天带伞吗 | 预测 | 预测 = 根据过去的规律来猜未来；预测有不确定性；信息越多，预测越准确 |
| 第16课 | 背答案的鹦鹉 | 理解vs背答案 | 背答案 ≠ 真正理解；过拟合 = 只会做见过的题，不会做新题；真正学会 = 理解规律，能举一反三 |
| 第17课 | 学校里的AI同学 | 训练集与测试集 | 训练集 = 学习用的练习题；测试集 = 考试用的新题 |

### 第三部分：机器学习（第18-24课）

| 课次 | 标题 | 主题 | 核心知识点 |
|------|------|------|------------|
| 第18课 | 小狗学握手 | 强化学习 | 强化学习 = 做对了得奖励，下次还这么做；学习循环：试一试→得反馈→再调整 |
| 第19课 | 贴标签的老师 | 监督学习 | 监督学习 = 有老师教、有答案的学习；标签越清楚，学习效果越好；标签错误则结果错误 |
| 第20课 | 没有标签的玩具箱 | 无监督学习 | 无监督学习 = 没有标签，自己找规律；聚类 = 把相似的东西分成一组 |
| 第21课 | 点亮聪明灯 | 神经网络基础 | 神经元 = 一个小灯泡，接收信号决定亮不亮；神经网络 = 很多神经元连在一起 |
| 第22课 | 从边边角角到整张脸 | 神经网络分层 | 分层处理：先看边缘→再看局部→最后拼成整体；分层让复杂任务变得简单 |
| 第23课 | 音量旋钮决定谁更重要 | 权重 | 权重 = 每个输入的重要程度；旋钮调大 = 这个输入更重要；多个输入加权加起来决定结果 |
| 第24课 | 慢慢调准的小箭头 | 梯度下降 | 梯度下降 = 一步一步找到最优；AI学习：猜答案→检查→小调整→越来越准 |

### 第四部分：大语言模型与AI应用（第25-32课）

| 课次 | 标题 | 主题 | 核心知识点 |
|------|------|------|------------|
| 第25课 | 会接话的鹦鹉2.0 | 语言模型 | 语言模型 = 预测下一个词的概率；AI有时会生成"听起来对但实际错"的内容——幻觉 |
| 第26课 | 给AI的清楚指令 | 提示词工程 | 模糊的指令 = 模糊的结果；清晰的指令 = 清晰的结果；提示词工程 = 写好指令的学问 |
| 第27课 | 从乱点点到清图画 | 生成式AI | 生成式AI是"一步一步"生成内容的；学习数据质量影响生成结果 |
| 第28课 | 会编故事的机器人 | AI的信息来源 | AI可能产生"幻觉"；要学会核实信息来源，不轻信一切 |
| 第29课 | 不要随便给陌生人 | 隐私保护 | 个人信息（地址、电话、姓名）是隐私，不能随便告诉别人；保护隐私是保护自己的安全 |
| 第30课 | 同样努力却分数不同 | 偏见 | 偏见 = 不公平的偏好；AI也可能因学习数据不公平而产生偏见 |
| 第31课 | 超级搭档 | 人机协作 | AI擅长快速处理大量数据；人类擅长判断意义和价值；AI+人类协作，比各自单独做更厉害 |
| 第32课 | 小小AI研究员毕业啦 | AI学习过程 | 学习会遇到困难和挫折；回忆帮助想起知识；知识是慢慢积累的；毕业不是终点，是新开始 |

## 知识点脉络图

```
信息是什么
    ↓
0和1（数据表示）
    ↓
编码 → 存储 → 像素 → RGB颜色
    ↓
压缩 + 噪声
    ↓
条件判断
    ↓
算法基础：循环 → 任务分解 → 调试
    ↓
分类 + 特征 + 预测
    ↓
理解vs背答案 + 训练集/测试集
    ↓
机器学习：强化学习 → 监督学习 → 无监督学习
    ↓
神经网络：神经元 → 分层处理 → 权重 → 梯度下降
    ↓
语言模型 → 提示词工程 → 生成式AI
    ↓
AI伦理：幻觉、信息来源、隐私、偏见
    ↓
人机协作
    ↓
总结：学习是终身过程
```

# 第4课《仓库管理员机器人》分镜稿

> **非常重要，必须严格遵守：对照上传的故事板图片进行创作，角色形象和画面风格必须以上传的图片作为关键帧画面。**


## 基础设定

- 年龄：3-9岁
- 时长：约3分钟
- 比例：16:9
- 主题：存储 = 把信息放进格子里
- 核心知识：存储 = 把信息放进格子里；格子 = 记忆位置；容量 = 格子的数量；压缩 = 把同样多的信息装得更小

## 角色与风格

- 团团：7岁男孩，黑灰短发，圆脸腮红，橙色上衣，蓝色背带裤，小书包，好奇活泼。
- 点点：白色云朵形AI机器人，深色屏幕脸，青蓝发光眼睛，胸口屏幕，头顶紫色天线。
- 风格：儿童水彩绘本动画风，清透明亮，角色和场景与上传参考图一致。
- 负面：不要写实摄影、3D塑料感、暗黑恐怖、角色变形、多余手指、文字乱码、画面拥挤、低清晰度。

- 旁白要求：旁白讲话时，团团和点点的嘴不要动，不要做对口型或配合讲话动作；只保持倾听、观察或轻微点头即可。

- 配音要求：发音清晰，全部使用标准普通话。

# 第18课《小狗学握手》上集分镜稿

> **非常重要，必须严格遵守：关键帧画面风格必须以上传的图片作为关键帧画面；人物形象必须跟人物角色板保持一致，包括多视角、表情、服装、身体比例、局部细节和角色气质。**

## 基础设定
- 年龄：3-9岁
- 时长：约1分32秒
- 比例：16:9
- 主题：强化学习 = 通过奖励来学会行为
- 核心知识：强化学习 = 做对了得奖励，下次还这么做；学习循环：试一试 → 得反馈 → 再调整；奖励机制是强化学习的关键；人类学习也用到强化学习
- 故事：团团和点点进入《小狗学握手》的情境，通过一个具体问题、一次动手观察和一次清楚总结，把本课知识变成小朋友能看懂、能复述的小故事。

## 角色与风格
- 团团：7岁男孩，黑灰短发，圆脸腮红，橙色上衣，蓝色背带裤，小书包，好奇活泼。
- 点点：白色云朵形AI机器人，深色屏幕脸，青蓝发光眼睛，胸口屏幕，头顶紫色天线。
- 本课新增角色/场景：沿用原分镜中的新增角色和场景设定，造型温和、可爱、适合3-9岁儿童观看。
- 风格：儿童水彩绘本动画风，清透明亮，高清质感，色彩饱和，减少模糊笔触，角色与上传参考图一致。
- 负面：不要写实摄影、3D塑料感、暗黑恐怖、角色变形、多余手指、文字乱码、画面拥挤、低清晰度。

- 旁白要求：旁白讲话时，团团和点点的嘴不要动，不要做对口型或配合讲话动作；只保持倾听、观察或轻微点头即可。
- 配音要求：发音清晰，全部使用标准普通话。

## 分镜

### 01. 故事开场 00:00-00:14
画面：公园草地，小狗汪汪正在和主人玩球，团团点点蹲在旁边看。
台词/旁白：旁白：今天，团团和点点来到公园，发现了一个关于"强化学习"的小秘密。
关键帧：公园的绿草地青翠茂盛，远处有几棵大树和蓝天白云；一只棕色小狗蹲在草地上，面前滚着一个彩色小球，尾巴摇得欢快；小狗主人站在一旁，脸上带着温和的笑容；团团蹲在草地上，眼睛一眨不眨地盯着小狗，满脸好奇；点点蹲在团团旁边，屏幕脸微微侧向小狗，天线轻轻晃动。

### 02. 今天学什么 00:14-00:28
画面：点点屏幕亮起强化学习主题图标，团团指向画面中的小狗，背景是公园草地。
台词/旁白：点点：第18课《小狗学握手》，我们要认识：强化学习 = 通过奖励来学会行为。旁白：别急，我们从一个看得见的小故事开始。
关键帧：点点的胸口屏幕亮起，显示一个骨头图标和一个金色星星图标并排闪烁；团团伸出一根手指指向草地上的小狗，表情认真；点点的屏幕上同时显示一个循环箭头图标，从"试"到"奖励"再回到"试"；公园的树叶被微风吹得轻轻摇晃，光影在草地上摇曳；小狗好奇地转头看向团团和点点。

### 03. 第一个发现 00:28-00:44
画面：主人伸出空手，小狗舔了舔但没反应；主人手里拿着饼干，小狗立刻握手。
台词/旁白：团团说："主人伸出手，小狗怎么不握手呀？"点点说："因为没有奖励！"旁白：把看不见的知识，变成看得见的图案，就更容易懂了。
关键帧：主人蹲下身，空着右手伸向小狗，小狗歪着脑袋凑过去舔了舔主人的手心，但没有任何反应；主人的左手藏在身后，手里握着一块小狗饼干；主人把饼干展示给小狗看，小狗立刻伸出爪子搭在主人手上；团团看着这一幕，眼睛越睁越大；点点的天线微微发光，屏幕显示一个问号变成感叹号。

### 04. 动手试一试 00:44-01:00
画面：团团拿饼干训练小狗，一遍遍练习。团团蹲在草地上，手里拿着小饼干，认真地训练小狗。
台词/旁白：团团：我来试试看！旁白：动手摆一摆、数一数、比一比，知识就会变得更清楚。
关键帧：团团蹲在草地上，手里拿着一块小饼干，伸向小狗；小狗犹豫了一下，闻了闻饼干，然后主动伸出爪子搭在团团的手上；团团开心地笑了，把饼干递给小狗；点点的屏幕显示一个金色星星图标，旁边写着"奖励！"；小狗吃完饼干，尾巴摇得更欢了，期待地看着团团手里还有没有更多饼干。

### 05. 规则出现 01:00-01:16
画面：揭示强化学习 = 给奖励让AI学会。点点的屏幕显示强化学习的公式和循环图。
台词/旁白：点点指着屏幕："这就是强化学习！做对了就奖励，AI慢慢就学会了！"团团说："原来奖励这么重要！"旁白：方法很重要，用对方法，问题就不再乱糟糟。
关键帧：点点的胸前屏幕放大显示，左边显示一个循环图：试一试 → 得奖励 → 再调整 → 试一试，循环箭头首尾相连；右边显示一个公式："强化学习 = 做对了得奖励，下次还这么做"；团团蹲在旁边认真看着点点的屏幕，眼睛亮了起来；小狗主动伸出爪子搭在团团手上，期待地摇着尾巴；点点的天线发出认可的淡紫色光芒。

### 06. 换一种方式看 01:16-01:32
画面：奖励不同，学到的行为也不同。团团尝试用不同东西奖励小狗，小狗学会了不同的动作。
台词/旁白：团团：我明白啦！用饼干教，小狗学会握手；用球教，小狗学会捡球！旁白：奖励不一样，学到的东西也不一样。
关键帧：团团手里拿着一个小球，展示给小狗看；小狗盯着球，团团把球扔出去，小狗飞奔过去捡回来；团团摸摸小狗的头作为奖励，小狗开心地舔团团的手；点点的屏幕显示两个并排的图标：左边是握手（强化学习），右边是捡球（强化学习），每个旁边都有对应的奖励图标；团团和小狗并肩蹲在草地上，团团脸上洋溢着成就感的笑容。

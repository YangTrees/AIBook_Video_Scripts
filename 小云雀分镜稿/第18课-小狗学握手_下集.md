# 第18课《小狗学握手》下集分镜稿

## 基础设定
- 年龄：3-9岁
- 时长：约1分28秒
- 比例：16:9
- 主题：强化学习 = 通过奖励来学会行为
- 核心知识：强化学习 = 做对了得奖励，下次还这么做；学习循环：试一试 → 得反馈 → 再调整；奖励机制是强化学习的关键；人类学习也用到强化学习

## 角色与风格
- 团团：7岁男孩，黑灰短发，圆脸腮红，橙色上衣，蓝色背带裤，小书包，好奇活泼。
- 点点：白色云朵形AI机器人，深色屏幕脸，青蓝发光眼睛，胸口屏幕，头顶紫色天线。
- 本课新增角色/场景：沿用原分镜中的新增角色和场景设定，造型温和、可爱、适合3-9岁儿童观看。
- 风格：儿童水彩绘本动画风，清透明亮，高清质感，色彩饱和，减少模糊笔触。
- 负面：不要写实摄影、3D塑料感、暗黑恐怖、角色变形、多余手指、文字乱码、画面拥挤、低清晰度。
- 旁白要求：旁白讲话时，团团和点点的嘴不要动，不要做对口型或配合讲话动作；只保持倾听、观察或轻微点头即可。
- 配音要求：发音清晰，全部使用标准普通话。

## 分镜

### 01. 核心知识揭示 00:00-00:16
画面：点点站在小讲台前，屏幕显示循环图：手势→奖励→开心→再手势。循环图三个步骤：1.试一试 2.得反馈 3.再调整。循环箭头图：三个步骤首尾相连形成完整的圆。团团用手指跟着箭头方向移动。
台词/旁白：旁白：本课的核心知识是：强化学习 = 做对了得奖励，下次还这么做；学习循环：试一试 → 得反馈 → 再调整；奖励机制是强化学习的关键。
关键帧：点点站在圆形小讲台前，讲台上的发光屏幕显示一个圆形循环图；三个步骤用箭头连接：第一步是"试一试"图标（一只手），第二步是"得反馈"图标（一颗金色星星），第三步是"再调整"图标（循环箭头）；团团站在点点旁边，右手食指顺着箭头方向移动；循环图发出橙色的光，步骤图标依次闪烁。

### 02. 效果展示 00:16-00:32

台词/旁白：团团说："强化学习 = 通过奖励来学会行为！做得好得奖励，下次还这么做！"团团说："试一试→得反馈→再调整，是学习的关键循环！"
关键帧：驯兽师站在舞台中央举起手说"握手！"；小狗毫不犹豫地立刻伸出爪子搭在驯兽师手上，表情自信又开心；驯兽师递出一块小零食，小狗张嘴接住；画面旁边出现一颗大金色星星和"成功！"文字，星星发出耀眼光芒；舞台灯光变得更明亮，红色幕布在微微飘动。

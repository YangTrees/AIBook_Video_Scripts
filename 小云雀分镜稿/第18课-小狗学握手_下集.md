# 第18课《小狗学握手》下集分镜稿
> **非常重要，必须严格遵守：以上传的图片作为关键帧画面；人物形象必须跟人物角色板保持一致，包括多视角、表情、服装、身体比例、局部细节和角色气质。不要生成新的角色和场景，都用我指定的参考图。**
## 基础设定
- 年龄：3-9岁
- 时长：约1分30秒
- 比例：16:9
- 主题：强化学习 = 通过奖励来学会行为
- 核心知识：强化学习 = 做对了得奖励，下次还这么做；学习循环：试一试 → 得反馈 → 再调整；奖励机制是强化学习的关键；人类学习也用到强化学习
- 故事：团团和点点进入《小狗学握手》的情境，通过一个具体问题、一次动手观察和一次清楚总结，把本课知识变成小朋友能看懂、能复述的小故事。
## 角色与风格
- 团团：7岁男孩，黑灰短发，圆脸腮红，橙色上衣，蓝色背带裤，小书包，好奇活泼。【角色：团团】
- 点点：白色云朵形AI机器人，深色屏幕脸，青蓝发光眼睛，胸口屏幕，头顶紫色天线。【角色：点点】
- 本课新增角色/场景：沿用原分镜中的新增角色和场景设定，造型温和、可爱、适合3-9岁儿童观看。
- 风格：儿童水彩绘本动画风，清透明亮，高清质感，色彩饱和，减少模糊笔触，角色与上传参考图一致。
- 负面：不要写实摄影、3D塑料感、暗黑恐怖、角色变形、多余手指、文字乱码、画面拥挤、低清晰度。
- 旁白要求：旁白讲话时，团团和点点的嘴不要动，不要做对口型或配合讲话动作；只保持倾听、观察或轻微点头即可。
- 配音要求：发音清晰，全部使用标准普通话。
## 分镜
### 01. 规则核心：奖励机制 00:00-00:15
画面：【@图片1】。对比画面。当小狗再次做出"握手"时，天空中落下一颗大金星；当它想偷懒趴下时，金星消失。点点站在小讲台前，用机械手臂指向这一幕。团团（动作）：认真地看着对比画面，点头表示理解。点点（动作）：用手臂指向这一幕。
台词/旁白：点点：奖励机制，是强化学习的核心密码！做对了，奖励函数就会给它大大的"加分"；做错了，就没有奖励。通过算分，小狗就能在心里画出一张完美的通关地图。

### 02. 核心知识揭示 00:15-00:30
画面：【@图片2】。点点将大屏幕升起，屏幕上显示出结构清晰的水彩风公式图表：狗爪、饼干、五角星图标依次闪烁，三个小狗原地小幅动画。下方伴随着三个齿轮咬合的动画：尝试、反馈、强化。团团认真地看着图表点头。团团（动作）：认真地看着图表，笃定地点头。点点（动作）：将大屏幕升起，显示公式图表。
台词/旁白：核心知识记牢啦：强化学习不需要人类手把手去教每一个细节，而是让机器像小狗一样，在每一次"尝试、反馈、记录再调整"的循环中，自己找到最棒的解题方法！
### 03. 生活大延伸：人类的智能 00:30-00:45
画面：【@图片3】。点点的屏幕投射出三面魔镜UI，展示高密度的生活写照：镜1 UI（足球）：团团练习射门，踢进死角后全场欢呼，他记住了这个发力角度。镜2 UI（艺术）：团团练习弹琴，弹对了一段复杂的旋律，爸爸妈妈在后面高兴地鼓掌。镜3 UI （生活习惯）：团团主动把绘本摆放得整整齐齐，得到了一个精美的小勋章。三个UI跟随旁白依次发光，每个UI内的动画也跟着旁白同步播放。每次只有一个UI发光，播动画。团团（动作）：在各个场景中积极参与，脸上洋溢着自豪的笑容。点点（动作）：屏幕投射三面魔镜。
台词/旁白：不仅小狗和AI会这一招，我们人类也是这样长大的！踢球时进球的快乐、弹琴时美妙的声音、收玩具时得到的夸奖，这些"精神奖励"都在强化我们的好习惯，让我们越来越棒！
